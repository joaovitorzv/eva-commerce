## Getting Started
To get a local copy up and running follow these simple steps below.

### How to setup locally

1. Clone the repo
   ```sh
   git clone https://github.com/joaovitorzv/eva-commerce.git
   ```
2. Change to cloned directory
   ```sh
   cd eva-commerce
   ```
3. Install YARN or NPM packages
   ```sh
   yarn install
   ```
4. Run the project
   ```sh
   yarn start
   ```
